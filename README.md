# Digital Brochure v2
디지털 브로슈어 v2

***

## Mixin

`{{ globalTitle }}`을 사용하면 어느 .vue 파일에서든 지정한 브로슈어 타이틀을 불러올 수 있습니다.
타이틀은 `app.vue` 파일에서 지정할 수 있습니다.

## 사용하는 플러그인

```
npm swiper
npm install --save scratchcard-js // 커버 액션용
```
